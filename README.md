# Contact Management System

A console-based Contact Management System built in Java.

## Features
- Add new contacts with validation
- View all contacts
- Search contacts by name (case-insensitive)
- Update existing contacts
- Delete contacts
- Duplicate phone number prevention
- Email and phone validation using Regex

## Project Structure
src/
└── com.contact/
├── model/Contact.java
├── service/ContactService.java
└── main/Main.java

## How to Run
1. Install JDK 17 or above
2. Open project in IntelliJ IDEA
3. Run Main.java

## Technologies Used
- Java JDK 26
- ArrayList for in-memory storage
- Regex for input validation
- OOP - Encapsulation, POJO design
