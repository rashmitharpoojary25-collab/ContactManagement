# Learnings

## What I Built
A console-based Contact Management System using Java
with full CRUD operations and input validation.

## What I Learned
1. CRUD operations - Create, Read, Update, Delete
2. Encapsulation - private fields with getters and setters
3. ArrayList for storing objects in memory
4. Regex validation for email and phone numbers
5. Case-insensitive search using equalsIgnoreCase()
6. Duplicate prevention using loop check

## Hardest Part
Understanding how to validate email and phone
using Regex patterns.

## How I Fixed It
Studied Regex patterns and used matches() method
in Java to validate input before saving.

## Key Takeaway
Always validate user input before storing data.
Never trust what the user types!
